imgViewer2 Plugin - dynamic image size example + "crisp option"

This example demonstrates the widget with a dynamic image width. Zoom in and out using the mousewheel, +/- controls, double click/tap or hold the shift key and drag out a rectangle. Left mouseclick and drag to pan. On touch enable devices pinch gestures can be used to zoom in and out and tap and drag to pan around.

Resize the window and try again

Combined with the "crisp option". When the image is "oversized" by 20% a "CSS trick" is used to make the image more crisp

Toggle the "crisp option" and Zoom in more.
